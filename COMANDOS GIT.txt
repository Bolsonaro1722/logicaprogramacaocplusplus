COMANDOS GIT



 para sincronizar uma pasta criada do repositorio "git clone"( endereço em que está o repositório).

após a sincronização
clique na pasta recém sincronizada
na barrqa de endereço apagar tudo e digitar "cmd"
após o prompt de comando estar aberto digitar:

git init--- inicia o repositório.
git add .---adiciona todos os arquivos criados
git commit -m "comentario" --adiciona um comentario pra ser inserido no repositório
